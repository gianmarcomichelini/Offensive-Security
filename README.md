## Description

This course covers offensive security techniques in two areas: binary exploitation and web exploitation. Topics range from memory management and OS-level protections to OWASP Top 10 attacks, with a strong emphasis on hands-on lab work.

---

## Topics

**Memory Management**
- Stack, heap, GOT/PLT, dynamic memory
- OS-level mitigations: stack canaries, ASLR, DEP

**Binary Exploitation**
- Code reuse attacks: ROP, ret2libc, gadgets
- Heap exploitation
- Bypassing stack canaries and ASLR

**Web Exploitation**
- Injection attacks: command injection, SQL injection
- Path traversal
- Authentication/authorization bypass: IDOR, JWT attacks
- OWASP Top 10: XSS, XXE, and others
